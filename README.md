# Rinkit

🚀 Groceries in a Blink

Rinkit is a grocery delivery platform designed to provide fast and convenient delivery of daily essentials.

## Features

- 🛒 Online Grocery Shopping
- ⚡ Fast Delivery
- 📱 Mobile-Friendly Interface
- 💵 Cash on Delivery (COD)
- 📲 UPI Payments
- 🏦 Net Banking
- 🎁 Offers & Discounts
- 📦 Order Tracking

## Technology Used

- HTML
- CSS
- JavaScript

## Vision

To make grocery shopping simple, affordable, and accessible for everyone.

## Future Updates

- OTP Login
- Delivery Partner App
- Admin Dashboard
- Inventory Management
- Live Order Tracking
- Analytics Dashboard
